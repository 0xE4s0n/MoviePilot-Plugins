# 保持插件最新

- 安装并开启`插件自动更新`插件，每次重启会更新已安装插件最新版本。也可设置cron定时任务更新插件。
- 如未刷新到插件更新，或者插件更新版本未变，可用`插件强制重装`插件进行重装。
